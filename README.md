#### Development
*Clone Project*
```
git clone https://github.com/prasom/currency-convertor.git
```
```
cd currency-convertor
```
*Get dependencies*
```
flutter pub get
```
*Run device*
```
flutter run -d {your device}
```
#### Testing
*Unit testing*
```
flutter test
